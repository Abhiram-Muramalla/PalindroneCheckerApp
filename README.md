# PalindromeCheckerApp


A lightweight Java-based utility designed to identify palindromes—strings that read the same backward as forward (e.g., "radar" or "level"). This project demonstrates core programming logic, string manipulation, and clean code practices.

##  Features

* **Case-Insensitive Matching:** Correctly identifies "Madam" as a palindrome.
* **Whitespace Handling:** (Optional/Planned) Ignores spaces and punctuation for phrase-based checking.
* **User Interaction:** Simple console-based interface for real-time testing.

##  Topics Covered

This project serves as a practical implementation of the following concepts:

* **String Manipulation:** Using `StringBuilder`, `reverse()`, and `charAt()`.
* **Control Flow:** Implementation of loops and conditional logic.
* **Algorithm Efficiency:** Comparing strings in $O(n)$ time complexity.
* **Input/Output:** Handling user data via the `Scanner` class.

##  Getting Started

### Prerequisites

* **Java Development Kit (JDK) 8** or higher.
* An IDE (IntelliJ IDEA, Eclipse, or VS Code) or a terminal.

### Running the App

1. **Clone the repository:**
```bash
git clone https://github.com/Abhiram-Muramalla/PalindroneCheckerApp.git

```


2. **Navigate to the project folder:**
```bash
cd PalindroneCheckerApp/App

```


3. **Compile the code:**
```bash
javac Main.java

```


4. **Run the application:**
```bash
java Main

```



##  Project Flow

1. **Start:** The application prompts the user for a string.
2. **Processing:** The app strips non-alphanumeric characters and converts the text to lowercase.
3. **Comparison:** The string is reversed and compared to the original.
4. **Result:** The app prints whether the input is a palindrome.

##  Use Case

This app is ideal for educational purposes, coding interviews, or as a utility module within a larger text-processing library.
